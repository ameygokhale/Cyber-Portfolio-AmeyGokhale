# Deleted Data Recovery Mechanisms for IoT Devices

This repository contains the implementation, documentation, and results of a university project focused on **recovering deleted data from IoT devices** using forensic tools such as **Autopsy** and **Foremost**.  
The project demonstrates data carving, file carving, memory analysis, and forensic imaging techniques.
