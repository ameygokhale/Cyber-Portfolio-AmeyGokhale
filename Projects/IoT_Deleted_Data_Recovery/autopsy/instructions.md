# Autopsy Instructions (summary)
This file summarizes the Autopsy steps used in the project (see Final_Report_Deleted_Data_Recovery.pdf for full detail and screenshots).

1. Use a write blocker when connecting the storage device to the forensic workstation.
2. In Autopsy: Create a new case -> Add Data Source -> Select the physical drive or disk image.
3. Choose destination image type (E01/RAW/SMART) and create a forensic image.
4. Verify image integrity using MD5/SHA256 (Autopsy can show verification results) or use `md5sum`/`sha256sum`.
5. Use 'Keyword Search' to locate deleted files by extension (.jpg, .png, .pdf, .txt).
6. Export recovered files to a secure folder for further analysis.
